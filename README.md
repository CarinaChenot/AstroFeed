# ASTROFEED

Get all the social feed from your favourite astronauts in one place !

### Features

* Retrieve twitter feed of each astronaut
* Display their position on the earth
* Cache data
* Drag and drop earth
* Display a selected tweet on the timeline

### Dev specs

* Twitter API
* Open notify API
* ES6
* SASS
* WebGL with three.js
* Git
* Gulp

### Installation

```Shell
cd space
npm install
cd dist
mkdir cache
cd ..
gulp
```
